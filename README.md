# <a href="https://stockmarketlandscape.herokuapp.com/"> Stock Market Landscape </a>

An interactive landscape generator app using the stock market.

#### Technologies:
+ javascript
    - node.js
    - p5.js
+ APIs
    - the color api
    - alpha vantage api
+ HTML and CSS
+ heroku
